DBFZ Bot is a Discord bot that parses JSON files of character movelists and displays them in Discord chat using embeds.

Invite the bot to your server using this link: https://discordapp.com/oauth2/authorize?client_id=418597940840628225&scope=bot&permissions=0

The bot is written in Python, and uses the Discord.py wrapper, BS4, FuzzyWuzzy, and DateUtils.

To run the bot, create a txt file in the same directory as DBFZ_Bot.py, paste your token in it and name it token.txt. Then execute DBFZ_Bot.py.
